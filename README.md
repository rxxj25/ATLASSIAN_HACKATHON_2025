## ğŸ”’ Privacy Policy

**Last Updated**: December 2025

### Overview
SprintPilot is an Atlassian Forge app that helps software teams visualize sprint data and generate retrospective reports.

### Data Collection and Storage

#### What Data We Store
- **Sprint Statistics**: Aggregated counts of issues (To Do, In Progress, Done) for the active sprint.
- **Sprint Metadata**: Sprint names, goals, and dates.
- **Confluence Page Links**: URLs to the retrospective reports generated by the app.

#### How We Store Data
All data is processed in-memory during execution. Any persistent data (like settings) is stored using **Atlassian Forge App Storage**, which is:
- Hosted and secured by Atlassian.
- Encrypted at rest.
- Accessible only to the app and authorized users.
- Compliant with Atlassian's security standards.

### Data Retention
- Data is strictly scoped to your instance.
- No data is stored in external systems or third-party servers.

### Personal Data
**We do NOT store personal data** in our own systems. All data storage is handled by Atlassian's Forge platform infrastructure.

### Data Access
- **Permissions**: The app requests `read:jira-work`, `read:sprint:jira-software`, `write:page:confluence`, and `write:space:confluence`.
- **Scope**: Data is scoped to your Jira/Confluence site and never shared across different sites.
- **No Third-Party Access**: We do not sell or share your data.

### Security
- All data transmission is encrypted (HTTPS).
- Data storage uses Atlassian's secure infrastructure.
- No external API calls are made with your data.
