# Privacy Policy for Code Review Assistant

**Last Updated**: November 2024

## Overview

Code Review Assistant is an Atlassian Forge app that helps software teams link Pull Requests to Jira issues and track code review progress.

## Data Collection and Storage

### What Data We Store

- **Pull Request Information**: PR URLs, titles, and review status linked to Jira issues
- **Review Status**: Code review status updates and timestamps
- **Issue Data**: Basic Jira issue information (issue keys, summaries)

### How We Store Data

All data is stored using **Atlassian Forge App Storage**, which is:
- Hosted and secured by Atlassian
- Encrypted at rest
- Accessible only to the app and authorized users
- Compliant with Atlassian's security standards

### Data Retention

- Data is stored as long as the app is installed on your Jira site
- You can delete data by unlinking PRs or uninstalling the app
- No data is stored in external systems

## Personal Data

**We do NOT store personal data** in our own systems. All data storage is handled by Atlassian's Forge platform infrastructure.

## Data Access

- Only users with access to the Jira issue can view PR information linked to that issue
- Data is scoped to your Jira site and not shared across sites
- No third-party access to your data

## Your Rights

You have the right to:
- View all PR links associated with your issues
- Delete PR links at any time
- Uninstall the app, which removes all associated data

## Security

- All data transmission is encrypted (HTTPS)
- Data storage uses Atlassian's secure infrastructure
- No external API calls are made with your data

## Changes to This Policy

We may update this privacy policy from time to time. The "Last Updated" date at the top indicates when changes were made.
